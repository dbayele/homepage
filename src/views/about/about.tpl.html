About

Hi! I'm a software developer in the Seattle Area. I started writing code when I was very young, but I have been developing software professionally for 2 1/2 years

Most of my career has been spent in developing web applications. Though technology changes rapidly, some things are always true. These values inform every project I work on.

User Focused

Applications are built for users not businesses. Whether they are internal or external, customers should always have a pleasing and effective user experience.

Responsive

Applications should be usable across a number of devices, form factors and displays. The best way to accomplish this is to use a mobile first design and responsive technology.

Maintainable

Applications should be built to be maintained. Not only does this save money it allows creative freedom throughout the application lifecycle.
